# OC-DAJR_P12_SportSee

# Instalation

- git clone this this repository
- install the dependencies
```bash
npm install
```
- start the server 
```bash
npm run dev
```

use the backend's [repository](https://github.com/OpenClassrooms-Student-Center/P9-front-end-dashboard) to start the api.
